## SDK installation

Follow these steps to get started:

### 1. Install Arcjet Redact

In your project root, run the following command to install the Arcjet redact library:

<Tabs>
  <TabItem label="npm">```bash npm i @arcjet/redact ```</TabItem>
  <TabItem label="yarn">```bash yarn add @arcjet/redact ```</TabItem>
  <TabItem label="pnpm">```bash pnpm i @arcjet/redact ```</TabItem>
</Tabs>

### 2. Redact some text

Now all you have to do to Redact text is to call the `redact` function. It can be
configured to redact a number of built-in entity types, or else you can provide
a custom detect function to redact your own types.

<Code code={QuickStartRedact} lang="ts" />

### 3. Unredact a response

If you want to un-redact some text that contains the replacements from the redact
function just call `unredact`, it is passed as the value in the return array.
