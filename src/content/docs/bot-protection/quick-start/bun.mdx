## Quick start

This guide will show you how to protect your entire Bun application from
bots and automated clients.

### 1. Install Arcjet

In your project root, run the following command to install the SDK:

{/* prettier-ignore */}
```sh
bun add @arcjet/bun
```

### 2. Set your key

[Create a free Arcjet account](https://app.arcjet.com) then follow the
instructions to add a site and get a key. Add it to a `.env.local` file in your
Bun project root.

Arcjet only accept non-local IP addresses with a fallback to 127.0.0.1 in development mode, and since Bun
doesn't set `NODE_ENV` for you like Node.js does, you also need to set
`ARCJET_ENV` in your environment file.

```ini title=".env.local"
# Run Arcjet in development mode because Bun.sh doesn't explicitly
# set an environment variable like Node.js does
ARCJET_ENV=development
ARCJET_KEY=ajkey_yourkey
```

### 4. Start server

{/* prettier-ignore */}
<Tabs>
  <TabItem label="TS">
  ```sh 
  bun run --hot index.ts
  ```
  </TabItem>
  <TabItem label="JS">
  ```sh 
  bun run --hot index.js
  ```
  </TabItem>
</Tabs>

Make a `curl` request from your terminal to your application. You should see a
`403 Forbidden` response because `curl` is considered an automated client by
default. See [the reference
guide](/bot-protection/reference/bun#removing-bot-detection-rules) for how to
configure this.

```shell
curl http://localhost:3000
```

The requests will also show up in the [Arcjet
dashboard](https://app.arcjet.com).
