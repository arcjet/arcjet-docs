### 4. Start server

Start the Flask dev server:

```sh
uv run flask --app main run --debug
```

Then make some requests to hit the rate limit:

```sh
for i in $(seq 1 5); do curl http://localhost:5000; done
```

After a few requests you should see a `429 Too Many Requests` response because
each request consumes 5 tokens from a bucket with a maximum capacity of 10.
