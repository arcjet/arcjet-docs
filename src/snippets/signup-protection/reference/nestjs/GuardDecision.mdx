Arcjet can be integrated using a global [guard](https://docs.nestjs.com/guards)
or per route guards. However these can't be used for signup protection rules
because you can't access the decision to handle the form submission. Instead,
you call the `protect` function in your route handler. See
<Link.Page href="/bot-protection/quick-start">bot protection</Link.Page> for an example where guards are
used.
