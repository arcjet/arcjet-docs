Add your key to a `.env` file in your project root.
You can also set
[`ARCJET_ENV`](https://docs.arcjet.com/concepts/environment#arcjet-env)
and other values in `.env` files.

```ini title=".env"
# Run Arcjet in development <https://docs.arcjet.com/concepts/environment#arcjet-env>.
ARCJET_ENV=development
# Your Arcjet Cloud API key <https://docs.arcjet.com/concepts/environment#arcjet-key>.
ARCJET_KEY=ajkey_yourkey
```
