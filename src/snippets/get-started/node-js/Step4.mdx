## 4. Start app

{/* prettier-ignore */}
<SelectableContent client:load syncKey="language">

  <div slot="JS" slotIdx="2">
    ```sh
    node --env-file .env.local index.js
    ```
  </div>

Visit [`http://localhost:8000`](http://localhost:8000) in your browser and
refresh a few times to hit the rate limit.

Wait 10 seconds, then run:

```sh
curl -v http://localhost:8000
```

The wait is necessary because the decision is cached for your IP based on the
`interval` configuration.

You should see a `403` response because `curl` is considered a bot by default
(<Link.Page href="/bot-protection/identifying-bots">customizable</Link.Page>).
