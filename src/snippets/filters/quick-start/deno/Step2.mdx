Deno loads environment variables with the [`--env-file` flag][deno-env].
Create a `.env` file in the root of your project and add your key.
You can also set
[`ARCJET_ENV`](https://docs.arcjet.com/environment#arcjet-env)
and other values in `.env` files.

```ini title=".env"
# Run Arcjet in development <https://docs.arcjet.com/environment#arcjet-env>.
ARCJET_ENV=development
# Arcjet key for your site (from <https://app.arcjet.com>).
# More info: <https://docs.arcjet.com/environment#arcjet-key>.
ARCJET_KEY=ajkey_yourkey
```

[deno-env]: https://docs.deno.com/runtime/reference/env_variables/
