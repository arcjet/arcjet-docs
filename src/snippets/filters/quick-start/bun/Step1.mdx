```sh
bun add @arcjet/bun
```

:::note
The package `@arcjet/bun` is made for Bun’s own [HTTP server][bun-http-server].
If you use Bun’s [Node.js compatibility][bun-nodejs-apis] layer,
use [`@arcjet/node`][arcjet-filters-node-js] instead.
:::
