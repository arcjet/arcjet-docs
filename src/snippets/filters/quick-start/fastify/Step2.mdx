Node.js loads environment variables with the [`--env-file` flag][node-env].
Arcjet integrations read from `ARCJET_*` environment variables.
Create a `.env.local` file in the root of your project and add your key:

```ini title=".env.local"
# Run in development mode; leave this unset in production.
ARCJET_ENV=development
# Get your site key at <https://app.arcjet.com>.
ARCJET_KEY=ajkey_yourkey
```

[node-env]: https://nodejs.org/docs/latest/api/cli.html#--env-filefile
