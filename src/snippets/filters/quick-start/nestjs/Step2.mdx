NestJS loads environment variables with the [`@nestjs/config`][nestjs-config]
module.
Arcjet integrations read from `ARCJET_*` environment variables.
Create a `.env` file in the root of your project and add your key:

Add your key to a `.env` file in your project root.

```ini title=".env"
# Run in development mode; leave this unset in production.
ARCJET_ENV=development
# Get your site key at <https://app.arcjet.com>.
ARCJET_KEY=ajkey_yourkey
```

[nestjs-config]: https://docs.nestjs.com/techniques/configuration
