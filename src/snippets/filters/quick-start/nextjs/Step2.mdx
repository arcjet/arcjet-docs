Next.js automatically loads environment variables from [certain files][next-env].
Arcjet integrations read from `ARCJET_*` environment variables.
Create a `.env.local` file in the root of your project and add your key:

```ini title=".env.local"
# Get your site key at <https://app.arcjet.com>.
ARCJET_KEY=ajkey_yourkey
```

[next-env]: https://nextjs.org/docs/pages/guides/environment-variables#environment-variable-load-order
